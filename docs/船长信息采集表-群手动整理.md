# 船长信息采集表（群内手动整理用）

> 用途：通过公开二维码进群后，**人工**整理群里公开分享的船长/船只信息，
> 便于后续联系船长取得授权、邀请入驻或录入演示数据。
>
> ⚠️ 合规要点：
> - 仅记录群内**主动公开**的船名、船型、出海信息等经营信息
> - 手机号/微信号等联系方式，建议先私聊取得本人同意再记录和使用
> - 录入 app 前务必取得船长明确授权（口头或书面）
> - 切勿自动化批量抓取群消息，仅人工摘抄

## 标准化字段说明（便于后续录入数据库）

- **船型**：从 {出海艇, 近海艇, 筏钓艇, 休闲渔船, 专业钓艇, 其他} 选最接近的
- **常出海域**：尽量结构化，如「东海/舟山」「黄海/青岛」「南海/深圳」
- **主打鱼种**：用 `/` 分隔，如「真鲷/黑鲷/石斑」
- **出海频率**：自由文本，如「每周2-3次」「仅周末」「禁渔期休」
- **价格区间**：填「散客价/座」或「包船价/天」

## 进群获客的标准动作（推荐流程）

1. **进群观察**（1-2天）：了解群内氛围、哪些船长活跃、有无晒渔获习惯
2. **建立信任**：适当互动、请教问题，别一进群就发广告
3. **私聊触达**：挑活跃船长私聊，介绍平台定位和入驻收益
4. **取得授权**：明确告知「想把您的船信息放到XX平台展示，方便钓友预约，是否同意？」
5. **录入或邀请**：船长同意后，要么你代为录入（演示期），要么发注册链接让他自己注册（长期）
6. **回访维护**：上线后定期同步展示效果，邀请他更新渔获照片
